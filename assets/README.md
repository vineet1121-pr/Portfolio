# Assets Folder

This folder should contain your portfolio images:

## Required Images:

1. **profile.jpg** - Your professional profile photo
   - Recommended size: 400x400px
   - Format: JPG or PNG

2. **project1.jpg** - Screenshot or image of your first project
   - Recommended size: 800x600px
   - Format: JPG or PNG

3. **project2.jpg** - Screenshot or image of your second project
   - Recommended size: 800x600px
   - Format: JPG or PNG

4. **project3.jpg** - Screenshot or image of your third project
   - Recommended size: 800x600px
   - Format: JPG or PNG

## Tips:

- Use high-quality images
- Optimize images for web (compress them to reduce file size)
- Use tools like [TinyPNG](https://tinypng.com/) or [Squoosh](https://squoosh.app/) to compress
- Maintain consistent aspect ratios for project images
- Use descriptive file names

## Optional:

You can add more images for additional projects or use placeholder images from:
- [Unsplash](https://unsplash.com/)
- [Pexels](https://pexels.com/)
- [Lorem Picsum](https://picsum.photos/)
